# arxiv-paper-chatbot
[Demo in hf](https://huggingface.co/spaces/skspark/arxiv-paper-chatbot)

## Prerequisites
```shell
pip install pdm
pdm install
```

## Usage
### Console
```shell
python app_console.py -k <openai-key> -g <language> -i <arxiv-file-id>
```
### Gradio
```shell
gradio app_gradio.py
```
